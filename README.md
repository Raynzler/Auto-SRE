# AutoSRE Platform

A production-like SRE demonstration platform showcasing observability, failure handling, and self-healing microservices.

## Features
- FastAPI service with Prometheus instrumentation
- Health checks and metrics endpoints
- Docker containerization
- (More coming: Grafana dashboards, chaos engineering, incident postmortems)

## Quick Start
```bash
docker-compose up --build
```

Access:
- API: http://localhost:8000
- Prometheus: http://localhost:9090
